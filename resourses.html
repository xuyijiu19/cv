Useful Resourses

  • World Wide Web Consortium (W3C)

    • An international community that develops open standards to ensure the long-term growth of the Web.

  • W3Schools

    • The largest web developer site on the internet. A school for web developers, covering all the aspects of web development: HTML, CSS, JavaScript and many more.

  • DevDocs

    • Multiple API documentations.

  • MDN Web Docs

    • Learning site and references for web developers.

  • freeCodeCamp

    • Free learning site. You could build projects and earn certifications.

  • HTML Cheat Sheet

    • HTML Cheatsheet by Stanford University.

  • CodePen

    • Online code editor and front end web developer community.

  • Udemy

    • Many courses on various topics including the web. Most are paid but discounted at a low price.

  • StackOverflow

    • Search for any questions, 99% it was already answered by someone on this website.

  • YouTube

    • Lots of free tutorials.
